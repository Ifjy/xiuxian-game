# 🧘‍♂️ 文字修仙 - 完整CMD风格挂机游戏

一个纯CMD风格的完整文字修仙游戏，包含角色创建、多维度属性系统、功法系统、宠物系统等丰富内容。

## 🎮 游戏特色

### 🎨 全新CMD风格设计
- **纯ASCII界面**：横竖线构建的游戏框架
- **三列布局**：小人居中，周围属性面板
- **修炼小人动画**：动态显示修炼状态
- **终端风格**：黑底绿字，复古体验
- **大型日志区域**：详细记录修炼历程

### 👤 完整角色创建系统
- **开局背景故事**：打字机效果的背景介绍
- **姓名与道号**：自定义你的身份
- **出身背景**：7种不同出身，各有加成
- **天赋点分配**：20点自由分配到四个属性

### 🎯 多维度属性系统

**基础属性**：
- **悟性**：影响功法学习速度和修炼速度
- **幸运**：影响事件触发和掉落品质
- **根骨**：影响修为上限和生命值
- **机缘**：影响奇遇和特殊事件概率

**战斗属性**：
- **攻击**：影响战斗伤害
- **防御**：减少受到的伤害
- **生命**：生命值上限

**成长属性**：
- 修为速度：受悟性、功法、宠物影响
- 突破概率：受悟性、功法影响
- 生命恢复：受功法影响

### 📚 功法系统

**功法等级**：
- **人阶功法**：基础功法，炼气期可学
- **地阶功法**：进阶功法，金丹期可学
- **天阶功法**：高级功法，化神期可学
- **仙阶功法**：顶级功法，渡劫期可学

**功法效果**：
- 修炼速度加成
- 攻击力提升
- 防御力提升
- 突破成功率提升
- 全属性提升

**功法升级**：
- 消耗灵石升级功法
- 提升功法效果
- 不同等级有不同效果加成

### 🐾 宠物系统

**宠物等级**：
- **凡阶宠物**：灵猫、灵犬
- **灵阶宠物**：飞鹰、灵蛇、仙鹤
- **圣阶宠物**：麒麟、凤凰
- **神阶宠物**：青龙、白虎

**宠物效果**：
- 幸运加成：增加掉落品质
- 寻宝能力：自动寻找物品
- 历练加成：增加历练收益
- 战斗加成：提升攻击/防御
- 修炼加成：增加修炼速度

### ⚔️ 核心玩法

1. **修炼系统**
   - 自动进行，每秒获得修为
   - 受悟性、功法、宠物影响
   - 打坐时会触发随机事件

2. **突破系统**
   - 修为达到上限时可突破
   - 受悟性、功法影响成功率
   - 失败会损失修为

3. **入定系统**
   - 3倍修炼速度，持续30秒
   - 与历练状态互斥
   - 冷却时间结束后可再次使用

4. **历练系统**
   - 历练期间修为不增长
   - 可获得修为、灵石、物品、宠物
   - 有失败风险

5. **炼丹系统**
   - 消耗灵石炼制丹药
   - 受悟性影响成功率
   - 可获得各种增益物品

6. **背包系统**
   - 查看和使用物品
   - 不同物品有不同效果

7. **商店系统**
   - 购买各种物品
   - 消耗灵石交易

   - 分配20点天赋值
4. 开始修仙之旅！

## 💾 存档管理详解

### ⚠️ 重要提示

**游戏使用浏览器 localStorage 存档，请务必了解以下限制：**

#### 存储位置
- **存储类型**：浏览器本地存储（localStorage）
- **存档文件**：
  - `xiuxian_save_1`：存档槽位1
  - `xiuxian_save_2`：存档槽位2
  - `xiuxian_save_3`：存档槽位3
- **存储位置**：浏览器用户数据目录

#### 不同浏览器的存档位置

**Windows系统：**
- Chrome: `C:\Users\用户名\AppData\Local\Google\Chrome\User Data\Default\Local Storage`
- Edge: `C:\Users\用户名\AppData\Local\Microsoft\Edge\User Data\Default\Local Storage`
- Firefox: `C:\Users\用户名\AppData\Roaming\Mozilla\Firefox\Profiles\xxx\webappsstore.sqlite`

**Mac系统：**
- Chrome: `~/Library/Application Support/Google/Chrome/Default/Local Storage`
- Firefox: `~/Library/Application Support/Firefox/Profiles/xxx`

### ⚠️ 存档丢失风险

1. **清除浏览器数据** → 所有存档丢失
2. **卸载浏览器** → 存档可能丢失
3. **使用隐私模式** → 存档不会保存
4. **换设备/浏览器** → 存档不互通

### ✅ 保护存档的方案

#### 方案1：使用游戏内导出功能（推荐）

1. 点击 **[存档]** 按钮
2. 点击 **[导出所有存档]** 按钮
3. 保存下载的 `.json` 文件
4. 需要时点击 **[导入存档]** 恢复

#### 方案2：手动备份localStorage

1. 按 F12 打开开发者工具
2. 进入 **Application** 标签
3. 左侧找到 **Local Storage**
4. 找到你的域名
5. 复制所有 `xiuxian_save_*` 的值保存

### 🌐 部署到服务器

#### 当前方案的特点

**优点：**
- ✅ 无需后端服务器，部署简单
- ✅ 可以托管到任何静态网站服务
- ✅ 无服务器成本

**缺点：**
- ❌ 玩家存档存储在各自浏览器中
- ❌ 换设备/浏览器会丢失存档
- ❌ 无法实现跨设备同步
