# 旧版本归档

此目录包含游戏重构前的单文件版本代码。

## 归档时间
2026-03-18

## 归档原因
游戏已从单文件架构重构为模块化架构（`js/` 目录）。

## 文件说明

### 核心文件
- **game.js** (321KB) - 旧的单文件游戏逻辑，包含所有系统
- **character-creation.js** - 旧的角色创建页面脚本
- **quest_methods.js** - 旧的任务方法模块

### 测试文件
- **test_combat_skills.js** - 战斗技能测试
- **test_suppression.js** - 镇压系统测试

## 新架构位置
重构后的代码位于：
- `js/main.js` - 新的入口文件
- `js/core/` - 核心模块
- `js/systems/` - 系统模块
- `js/config/` - 配置文件
